# Skill Nexus - Career Path Finder

**Skill Nexus** هو مشروع تفاعلي يهدف لمساعدة الطلاب المهتمين بالتقنية في تحديد مسارهم المهني المناسب من خلال اختبار مكون من 5 خطوات.

## 🎯 الفكرة (The Concept)
بناءً على اهتمامات المستخدم وقدراته التقنية، يقوم البرنامج بتحليل الإجابات واقتراح المسار الأنسب (مثل: Cybersecurity, UI/UX Design, Web Development).

## 🛠️ الأدوات المستخدمة (Tech Stack)
- **Programming Language:** Python
- **Logic:** Decision-making algorithms
- **Design Inspiration:** User-centric career mapping

## 📝 كيفية الاستخدام (How to use)
1. قم بتحميل الملف أو عمل Clone للمستودع.
2. قم بتشغيل الملف الرئيسي (Python script).
3. أجب على الأسئلة الخمسة لتكتشف مسارك التقني.

---
*تم تطوير هذا المشروع كجزء من تطوير أدوات التوجيه المهني التقني.*
